# 二叉树
- n ：总节点数    
n0：度为0的节点数   
n1：度为1的节点数   
n2：度为2的节点数   
n = n0 + n1 + n2    

- 二叉树的边数： n - 1    
二叉树的边数： n1 + 2*n2    

- n1 + 2*n2 = n - 1   
n1 + 2*n2 = n0 + n1 + n2  - 1   
n2 = n0 - 1
